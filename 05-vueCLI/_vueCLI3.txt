目录/文件			说明
dist				打包后的项目。
node_modules		npm依赖。
public				静态资源。不会打包，原封不动的复制。
					favicon.ico	网站图标。
					index.html	网站首页。
src					开发目录。
						assets		资源文件。可能会打包进js文件内，并且会重命名。
							css		样式文件。
								base.css		自己设置的全局样式。
								normalize.css	初始化全局样式。
							img		图片文件。
						common		公共js文件（公共常量、工具类、混入等）。
						components	公共组件。
							common		完全公共的组件（与当前项目无关）。
							content		与当前项目业务有关的公共组件。
						network		网络请求相关。
						router		路由。
						store		vuex。
						views		视图。
						App.vue		网站入口。
						main.js		项目入口。
.borwserslistrc		浏览器的兼容配置。
.editorconfig		文件样式配置。
.gitignore			git忽略文件。
package-lock.json	项目npm配置文件（绝对版本号）。
package.json		项目npm配置文件（大致版本号）。
README.md			项目说明。
vue.config.js		配置文件。
